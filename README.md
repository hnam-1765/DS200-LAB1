# Lab1 Movie Analysis

Thư mục bao gồm: 

- `python_solution/`: Mã nguồn python cho 4 bài
- `results/`: Các file kết quả được sinh ra từ dữ liệu `movies.txt`, `ratings_1.txt`, `ratings_2.txt`, `users.txt`.
- `screenshots/`: Ảnh kết quả cho từng bài.
- `tools/generate_results.py`: Tạo file kết quả


Cách chạy:

```bash
git clone https://github.com/hnam-1765/DS200-LAB1.git
cd /DS200-LAB1/python_solution
python3 run_all.py
```

## Tạo file kết quả
